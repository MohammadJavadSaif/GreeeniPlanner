// readme.html · v1.0.8

Plant a goal. Watch it branch.
Don't water it with vague intentions.

Most goal-tracking apps want you to feel like you're filling out a tax form. GreeniPlanner wants you to feel like you're running a small, well-organized, slightly intimidating garden — no accounts, no cloud, no "upgrade to Pro to see your own data."

01The Hierarchy

You plant a Long-term Goal at the top. It grows Medium-term and Short-term branches. Those branches grow Tasks. Tasks grow Sub-tasks. Five levels, top to bottom, no skipping steps.

Long-termRun a marathon by 35
└─ Medium-termSub-3:30 in the next 18 months
└─ Short-termFinish a half marathon this spring
└─ TaskBuild a 12-week training block
└─ Sub-taskBuy proper running shoes

Drag the ≡ handle that appears on hover to reorder anything — goals, nested goals, tasks, sub-tasks. Items can only be reordered within their own parent group.

02What Every Branch Holds
Name *
Required. Vague goals don't get a slot.
Description
Optional — but future-you will thank present-you.
Timeframe *
Three side-by-side boxes: Year / Month / Day. Fill whichever applies. Required for goals, optional ("Duration") for tasks.
Date
Single optional date for goals. Tasks get Start Date + End Date — set both and Duration auto-calculates.
Priority
Tasks only — High, Medium, or Chill.
Type (edit)
You can change a goal's type while editing. Downgrading cascades to nested items automatically.
03The Two Rooms
🗂️

Goals — the long game

The tree. Stats at the top count everything. Expanding cards stays expanded when you add items — no more collapsing on every save.

☀️

Daily Tasks — the right-now

A date strip at the top. Pick a day, see its tasks. Copy or Move tasks between days. Link tasks back to your Goals tree.

05What's New in v1.0.8
Size
~5MB installer, ~15MB installed. Rebuilt on Tauri — uses Windows' built-in WebView2 instead of bundling a full browser. 10–15× smaller than before.
Calendar
Click month or year to jump directly. All calendars across the app now have a month-grid picker and scrollable year picker — no more clicking ◀▶ through 50 months.
Timeframe
Year / Month / Day — three boxes, fill any. Replaced the old single number + dropdown. Tasks use optional "Duration" instead of required "Timeframe."
Tasks
Start Date + End Date for tasks and sub-tasks. Setting both auto-fills Duration. End date before start date is blocked with a clear error.
Reorder
Drag ≡ to reorder anything. Goals, nested goals, tasks, sub-tasks, daily tasks — all reorderable within their group. Order saves immediately.
Daily
Copy and Move tasks between days. Blue copy icon and amber move icon on every daily task row. Pick the target date from the in-app calendar.
Goals
Edit goal type + cascade downgrade. Change Long → Short while editing — nested items downgrade automatically to stay valid. Expanded cards no longer collapse when you add something.
Global
Gregorian / Shamsi / Hijri calendar toggle. GR / SH / HI buttons in the topbar. All date displays, month names, and date pickers adapt instantly. Data always stored as Gregorian internally.
Global
Persian, Arabic, Chinese fonts. Vazirmatn and Noto Sans SC loaded automatically. Type in any of these languages and it renders correctly.
Global
Styled dialogs. All confirm and alert popups now match the app's visual design — no more bare browser dialogs.
06Worth Knowing
🌓 Light / Dark — top-right corner, pick your contrast.
📅 Calendar system — GR / SH / HI in the topbar. Switch anytime; all calendars update immediately.
Everything autosaves. There is no save button because there is nothing to forget to click.
Deleting a branch deletes what's under it. GreeniPlanner will always ask first with a styled dialog.
Installing over a previous version is safe. All your data migrates automatically — nothing is deleted.
07Built Different, On Purpose

GreeniPlanner is unapologetically brutalist: hard edges, no rounded corners, no soft shadows pretending to be friendly. JetBrains Mono and Space Grotesk do the talking. Built with Tauri — a real native Windows app, not a browser tab in a frame. Starts fast. Stays fast.
