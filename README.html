Plant a goal. Watch it branch.
Don't water it with vague intentions.

Most goal-tracking apps want you to feel like you're filling out a tax form. GreeniPlanner wants you to feel like you're running a small, well-organized, slightly intimidating garden — no accounts, no cloud, no "upgrade to Pro to see your own data."

01 The Hierarchy

You plant a Long-term Goal at the top. It grows Medium-term and Short-term branches. Those branches grow Tasks. Tasks grow Sub-tasks. Five levels, top to bottom, no skipping steps — because "I'll just wing it" is how a ten-year goal quietly turns into a browser tab you never close.

Long-termRun a marathon by 35
└─ Medium-termSub-3:30 in the next 18 months
└─ Short-termFinish a half marathon this spring
└─ TaskBuild a 12-week training block
└─ Sub-taskBuy proper running shoes

Tasks and Sub-tasks are planted inside the tree, not from the top — they need a parent branch to grow from, same as in nature. Hit the + New Goal button for Long, Medium, or Short-term; use the small + buttons inside a card to grow Tasks and Sub-tasks underneath it.

02 What Every Branch Holds
Name *
Required. Vague goals don't get a slot.
Description
Optional, but future-you will thank present-you.
Timeframe *
A number, plus Years or Months.
Date
Optional, picked from the in-app calendar.
Priority
Tasks only — High, Medium, or Chill. Not everything is a fire.
03 The Two Rooms
🗂️

Goals — the long game

This is the tree. Stats at the top quietly count everything for you — long-term, medium-term, short-term, tasks, sub-tasks — so you always know how big your forest has gotten.

☀️

Daily Tasks — the right-now

This is not the tree. This is today. A date strip lets you flip through days like a desk calendar, jump a week, or leap anywhere via the calendar button.

You can link a daily task back to a Task from your Goals tree — so "answer client emails" can quietly trace back to "Launch the thing" three levels up, and you remember why you're doing it at 9am on a Tuesday.

04 Today's Moods

Each daily task gets one of four priorities, and they are not subtle:

Kill it!This is the thing. Today.
Mid Slash!Important, not on fire.
Final Rip!The last push to finish something.
Chill Bro!Low stakes. Breathe.
05 Worth Knowing
🌓 Light / Dark — top-right corner, pick your contrast.

Everything autosaves. There is no save button because there is nothing to forget to click.

Deleting a branch deletes what's under it. GreeniPlanner will always ask first, but it won't lie to you about the size of what you're cutting.

Your data lives only on this machine. Don't move or rename the app file once you start using it — there's no cloud sync to fall back on, the data is genuinely, only, right here.
06 Built Different, On Purpose

GreeniPlanner is unapologetically brutalist: hard edges, no rounded corners, no soft shadows pretending to be friendly. JetBrains Mono and Space Grotesk do the talking. It looks like a tool because it is one — not a wellness app cosplaying as a tool.
